## 📋 About This Project

This is a final project submission for Web Development. The website is a single-page responsive site for **Patel Bros Barber**, a traditional men's grooming shop established in 1998.

### ✅ Features
- Fully responsive (mobile + desktop)
- Services menu with Indian pricing (₹)
- 2x2 image gallery
- Google Maps embed (Vadodara location)
- Business hours and contact info
- External booking button
- Social media links
- No logo (text-based branding)

### 🛠️ Built With
- HTML5
- CSS3
- Bootstrap 5.3
- Font Awesome 6
- GitHub Pages (Hosting)
